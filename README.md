🎯 Цель проекта
Обеспечить равный доступ к информации о научных грантах для всех категорий пользователей, включая:

! Незрячих и слабовидящих (скринридер NVDA)

! Людей с нарушениями моторики (навигация только с клавиатуры)

! Пользователей с когнитивными особенностями (дислексия, СДВГ, сниженная концентрация внимания)

Результат: Интерактивный HTML-прототип трёх ключевых страниц, соответствующий требованиям WCAG и ГОСТ Р 52872-2019.

👥 Пользовательский сценарий
Целевое действие пользователя на портале:

!Использовать фильтры поиска — найти гранты по своей научной области

!Открыть карточку гранта — перейти к детальной информации

!Понять условия участия и сроки подачи — оценить релевантность и успеть подать заявку
